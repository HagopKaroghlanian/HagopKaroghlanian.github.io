Bootstrap Website

A Conference Website

This project is a pure Bootstrap website(With a touch of plain old CSS). It is fully responsive, with a great sign up feature using Bootstraps awesome features. Pressing on the "Register Now" Button triggers a registration form in which the user can input their personal information, as well as choose topics of interest from a tickbox. The user can also input their payment info and press the Register button at the end.
