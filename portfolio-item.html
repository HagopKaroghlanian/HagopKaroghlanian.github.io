Lake Sevan

Travel website

This is the very first project i worked on while going through Teamtreehouses' Front-End Web Development track. It's a static website using purely HTML/CSS. It was originally a website of Lake Tahoe in California, however i remade it using our wonderful Lake Sevan!

The project displays a large header image of Lake Sevan, with a beautiful title and a customized arrow to add style. Followed by some information about the Lake, the website continues to the first section that it about the wildlife. Again, i have attached a picture with a button under it. I customized the button with CSS color contrast to make it stand out.

The last section is divided between two containers using flexbox. I have made it responsive to different screen sizes taking into consideration good UX. At smaller screen sizes, the images line up one under the other, while at larger screen sizes, they flex across the screen
